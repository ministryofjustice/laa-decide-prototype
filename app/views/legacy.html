GOVUK Prototype Kit prototypes

[source code: https://github.com/ministryofjustice/laa-decide-prototype]

  • v1 - Figma prototype v4 @ 01/07/2021. Hard-coded data.

  • v2 - dynamic version of Figma prototype v4. Allows interaction with the service. Supported functionality:

    • Add to my list
    • Remove from my list
    • Merits decisions on emergency and substantial proceedings
    • Grant all
    • Refuse all
    • Save and come back later
    • Send back to provider
    Unsupported functionality:
    • Search - search bar is present but does not function
    • Filter - filter is present but does not function
    • Application history - hardcoded. Interacting with the service does not update the history

  • v3 - dynamic version of Figma prototype v5 - MVP. Includes all functionality from v2, fixes issues with Search and Application History, plus updates to bring it in line with the MVP version of v5 of the Figma prototype.

  • v4 - dynamic version of Figma prototype v5 - post-MVP Includes all functionality from v3, plus additional features to support post-MVP work. This includes:

    • Means assessments
    • Requests for more information
    • Changing merits assessment data

  • v5 The final version of the prototype before the Alpha was paused. Uncertain of specifics added between v4 & v5.
